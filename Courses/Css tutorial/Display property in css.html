Display Property in Css

The Css Display property sets whether an element is treated as a Block or Inline Element and the layout used for its children, such as Flow Layout , Grid and Flex Formally, the display property sets an element's inner and outer display types. The outer type sets an element's participation in flow layout; the inner type sets the layout of children. some value of display are fully defined in their own individual specifications; for example- the detail of what happenes when display: Flex is declared is defined in the css flexible Box Model specification.
